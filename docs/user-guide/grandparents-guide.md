# Grandparents Guide

## Who this is for

Use this guide for grandparents or other family members who mostly want to check progress without changing things.

The best setup for this is usually:

- role: `viewer`
- audience: `adult`

## What a viewer can do

- open home
- view the task board
- view projects
- view timeline
- view stats

A viewer cannot:

- log new tasks
- edit task details
- start or finish jobs
- change projects

## Why this is useful

It gives someone a clear picture of what is happening in the house without giving them controls they do not need.

Good examples:

- a grandparent checking how a room project is going
- a relative following shared household jobs
- someone who wants awareness without admin work

## What to open first

Start with:

- `/`: for the main home cards
- `/tasks`: to see the current board
- `/projects`: to see bigger work
- `/stats`: for a summary view

## Best setup advice

- use `viewer` if the person should not edit anything
- keep the home screen simple
- avoid giving setup access unless they will really use it
- if they only care about one property or part of the house, limit location access
