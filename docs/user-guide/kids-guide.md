# Kids Guide

## Who this is for

Use this guide for people set to `Under 12`.

The kids view is simpler on purpose.
It only shows jobs picked for that child.

## What kids see

- a home screen with `My jobs`
- a task view with jobs chosen for them
- simple buttons to start or finish a job

Kids do not see the full planning, project, and reporting screens.

## How to use it

1. Open home.
2. Tap `My jobs`.
3. Pick a job.
4. Tap `Start job`.
5. When it is finished, tap `I finished this`.

## What the labels mean

- `Ready to go`: you can start
- `In progress`: you already started
- `Finished`: you are done
- `Due today`: try to do this today
- `Needs attention`: this one has been waiting too long

## Tips for adults helping kids

- keep job titles short
- use simple notes
- assign only a few jobs at a time
- avoid putting project planning in the child view
- use the weekly wins count as encouragement
