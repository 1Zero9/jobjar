# Adults Guide

## Who this is for

Use this guide for adults who help run the household board.

That usually means:

- `admin`
- `power_user`
- `member`

## What adults can do

- capture work quickly on `/log`
- manage the shared task board on `/tasks`
- promote bigger work into projects on `/projects`
- review dates in `/projects/timeline`
- review progress in `/stats`

Admins can also use `/settings` and `/admin`.

## Normal daily flow

1. Notice a task or issue.
2. Open `/log`.
3. Add a short title and room.
4. Leave extra detail for later unless it matters now.
5. Open `/tasks` to assign, finish, or tidy the item.

## When to use projects

Turn a task into a project when the work:

- needs several steps
- has a budget
- needs materials
- has checkpoints
- has a target date

Projects work best when the parent item stays simple and the detail lives in project steps.

## Useful habits

- Keep capture fast.
- Use notes for context, not for the whole plan.
- Use milestones for meaningful checkpoints.
- Track materials that still need to be bought.
- Use location access if someone should only see one part of the household.

## If you are an admin

Use `/settings/people` to:

- set role
- set age group
- set theme
- limit location access

Use `/settings` and `/admin` for broader household setup.
