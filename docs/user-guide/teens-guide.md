# Teens Guide

## Who this is for

Use this guide for people set to the `12 to 18` age group.

The teen view keeps the main household board, but the wording and presentation are simpler than the adult view.

## What you will usually use

- `/`: home
- `/tasks`: your board
- `/log`: log a job if your role allows it
- `/projects`: bigger work

## The simplest way to use JobJar

1. Open home.
2. Tap `Task board`.
3. Pick a job.
4. Start it.
5. Finish it when done.

If something new needs to be recorded, use `Log a job`.

## What the board means

- `Open`: ready to do
- `In progress`: started but not finished
- `Done`: finished
- `Due today`: needs attention today
- `Lapsed`: overdue

## Projects

Projects are bigger jobs broken into smaller steps.

Examples:

- decorate a room
- clear a shed
- sort a bedroom

You do not need to understand the whole project plan to help.
Just open the project or task step that is assigned to you.

## Good habits

- finish one job before starting too many others
- use notes if something needs explaining
- keep titles short
- ask an adult before changing project plans or budgets
